Chromaplex is a collection of four distinct puzzle games in one beautifully consistent iOS app. Code Breaker gives you the deep, deductive satisfaction of classic Mastermind. Ball Sort, Binary Grid, and Flow Connect round it out with tactile, spatial puzzles that use a completely different part of your brain. One app, four genres, 930 handcrafted levels.

### Why This App

Most puzzle apps are either a single game stretched thin across hundreds of variants, or a grab bag of re-skinned clones with no unifying vision. Chromaplex takes the opposite approach: four genuinely different puzzle types, each designed as a first-class experience, linked by a shared progression system, a single daily challenge, and a consistent visual language.

### How It Works

Code Breaker asks you to crack a secret color code using feedback pegs and logic. Ball Sort asks you to sort colored balls into tubes without spilling. Binary Grid enforces Takuzu rules across grids up to 12x12. Flow Connect asks you to draw paths between matching endpoints that fill the entire grid. Each game tracks stars, completion, and streaks independently, but all roll up into a single progression on the home screen.

### Privacy-First by Design

Chromaplex stores all game data on-device. Progress, streaks, hints, daily challenge results, and lives never leave your phone. Optional Game Center leaderboards use anonymous identifiers managed by Apple. Rewarded ads are shown through Google AdMob only when you explicitly opt in to earn an extra life or hint, and they follow Apple's App Tracking Transparency framework.
